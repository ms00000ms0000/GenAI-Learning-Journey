# Day 1 - LLM Basics

## What is LLM?
A Large Language Model (LLM) is a neural network trained on massive amounts of text data to predict the next token and generate human-like responses.

## Tokens
Tokens are small units of text processed by an LLM. They can be words, subwords, punctuation marks, or characters depending on the tokenizer.

## Context Window
Context Window is the amount of information an LLM can remember and process at one time while generating responses.

## Temperature
Temperature controls randomness in responses. Lower values produce predictable answers while higher values generate more creative and diverse outputs.

## Hallucination
Hallucination occurs when an LLM confidently generates incorrect or fabricated information because it predicts text patterns instead of verifying facts.

## Key Learning
LLMs do not think like humans. They generate responses by predicting the most probable next token based on context.
