# Day 2 - Prompt Engineering

## Zero Shot Prompting
The model is given a task without examples and generates output using only its existing knowledge and training.

## One Shot Prompting
A single example is provided before the actual task so the model understands the expected pattern and format.

## Few Shot Prompting
Multiple examples are provided to help the model identify patterns and generate more accurate responses.

## Role Prompting
The model is assigned a role such as teacher, analyst, or software engineer to improve response quality and relevance.

## System Prompt
A system prompt defines the behavior, tone, and rules the model should follow during the conversation.

## Chain of Thought (CoT)
Chain of Thought encourages the model to solve problems step by step, improving reasoning and accuracy.

## Key Learning
Well-structured prompts significantly improve output quality and are essential for building effective GenAI applications.
