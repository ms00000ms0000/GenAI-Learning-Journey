# Day 4 - FAISS

## What is a Vector Database?
A Vector Database stores embeddings and allows efficient similarity searches using mathematical distance calculations between vectors.

## FAISS
FAISS (Facebook AI Similarity Search) is an open-source library used for storing and searching vector embeddings efficiently.

## Indexing
Indexing organizes vectors into optimized data structures that allow fast retrieval without scanning every vector.

## Nearest Neighbor Search
Nearest Neighbor Search finds vectors closest to the query vector and returns the most semantically relevant results.

## Top-K Retrieval
Top-K Retrieval returns the K most relevant chunks or documents based on similarity scores.

## Retrieval
Retrieval is the process of finding relevant information from a vector database before sending it to an LLM.

## Key Learning
FAISS enables fast semantic search and acts as the retrieval engine in many RAG applications.
